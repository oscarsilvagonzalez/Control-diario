⚙️ Configurar Google Sheets
Para que los datos se sincronicen automáticamente, necesitas crear un Apps Script una sola vez. Sigue estos pasos:
  1. Ve a script.google.com y crea un nuevo proyecto.
  2. Borra el contenido y pega el código que aparece abajo.
  3. En la primera línea, cambia el ID de tu hoja de Google Sheets (está en la URL de tu hoja entre /d/ y /edit).
  4. Haz clic en Implementar → Nueva implementación → Aplicación web. Pon acceso: Cualquier usuario.
  5. Copia la URL generada y pégala abajo.
const SHEET_ID = "TU_ID_DE_HOJA_AQUI"; function doPost(e) { const data = JSON.parse(e.postData.contents); const ss = SpreadsheetApp.openById(SHEET_ID); let sheet = ss.getSheetByName(data.hoja || "Registros"); if (!sheet) sheet = ss.insertSheet(data.hoja || "Registros"); if (sheet.getLastRow() === 0) { sheet.appendRow(data.cabeceras); } sheet.appendRow(data.fila); return ContentService .createTextOutput(JSON.stringify({ok:true})) .setMimeType(ContentService.MimeType.JSON); }
